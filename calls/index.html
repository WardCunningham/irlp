24 Hour Call History




About this Map

The lines on this map represent calls in progress on the irlp network of amateur radio stations. Black lines are station to station calls. Gray lines are party line calls placed through reflectors. Each frame shows the activity as reported in the irlp status plain text file which is sampled every 10 minutes. (Calls under 10 minutes can be missed.) The movie starts 24 hours ago, proceeds to the present, and then repeats indefinitly with progress marked by the relentless progression of the sun.

Stations are added to the map as they place calls. A station that does not report its location is plotted in the North Atlantic ocean. (Errors in our projection algorithm cause other stations to be plotted offset from their proper locations.)

The map is drawn with a java applet. If you don't see the map, perhaps you have java disabled or you are using an older web browser that does not support java version 1.2. The applet reads a movie script which is generated by the server from the currently available data. This applet is released as open source software under the gnu general public license version 2. It is one of many projects I've persued connecting amateur radio and computers.
